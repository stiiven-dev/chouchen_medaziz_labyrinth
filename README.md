# Jeu de Labyrinthe

## Description
Ce projet est un mini-jeu web où le joueur doit sortir d'un labyrinthe le plus rapidement possible.

Le personnage se déplace avec les flèches du clavier et doit atteindre la sortie.

## Technologies utilisées
- HTML
- CSS
- JavaScript
- DOM Manipulation
- LocalStorage

## Fonctionnalités principales
- Déplacement du personnage avec les flèches
- Détection des murs
- Chronomètre
- Score
- Meilleur temps sauvegardé localement
- Animations CSS
